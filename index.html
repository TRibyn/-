Руководство по FlutterFlow

Вы можете использовать данный интерактивный прототип как самостоятельный Telegram Mini App. Но если вы собираете его полноценно внутри FlutterFlow, используйте следующую архитектуру:

Шаг 1: Схема Firestore

Коллекция: estimates

  • tg_user_id (String)
  • created_at (Timestamp)
  • total_sum (Double)
  • adjustment_percent (Integer)
  • items_list (List of JSON Objects with name, price, qty, unit, category)
Шаг 2: Custom Action на Dart

Создайте во FlutterFlow Custom Action для обратной отправки данных в бота:

import 'dart:js' as js;
import 'dart:convert';

Future sendEstimateToTelegram(
  double totalSum,
  List<dynamic> itemsList,
) async {
  try {
    if (js.context.hasProperty('Telegram') && 
        js.context['Telegram'].hasProperty('WebApp')) {
      var webApp = js.context['Telegram']['WebApp'];
      
      var data = {
        'total': totalSum,
        'items': itemsList,
      };
      
      webApp.callMethod('sendData', [jsonEncode(data)]);
    }
  } catch (e) {
    print("Error sending data to Telegram: $e");
  }
}
Шаг 3: Настройка в Telegram
  1. Сделайте веб-деплой проекта во FlutterFlow.
  2. Перейдите в @BotFather и отправьте команду /newapp.
  3. Свяжите Mini App со своим ботом и укажите ссылку на деплой FlutterFlow.
